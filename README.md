# UD-CNN

Вообщем то :
- Должно быть примерно в 0.5 раза более эффективно чем DCNN.
- Не обещаю , но попытаюсь сократить время обучения как смогу.
- А также подбор идеальных значений для весов , а также параметров во время обучения.
